Implementation on opendata.swiss

The following documentation provides an information on how DCAT-AP CH is currently implemented on opendata.swiss and how it is “mapped” to further standards in the domain of geoinformation (GM03)

Name Version Format Description
Handbook opendata.swiss v1 html Guidelines for publishing data on opendata.swiss
DCAT-AP CH Implementation v1 html This page in the handbook describes the way the DCAT-AP CH is currently implemented on opendata.swiss. It helps data publishers to prepare their metadata, so that they will appear in the intend way on opendata.swiss
Mapping of Geodata v1 html This page in the handbook describes the mapping of geodata coming geocat.ch (based on the standard) GM03 to DCAT-AP CH as it is currently implemented on opendata.swiss.
